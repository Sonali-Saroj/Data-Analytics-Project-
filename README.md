# 📊 Sales Data Analysis Project

## 📌 Overview

This project focuses on analyzing sales data using **Python and SQL** to extract meaningful insights and support data-driven decision-making.
It includes data cleaning, querying, and visualization.

---

## 🛠️ Tech Stack

* Python (Pandas, NumPy)
* SQL (MySQL)
* Jupyter Notebook
* Excel / CSV Dataset

---

## 📂 Project Structure

```
project-folder/
│── data/               # Dataset files
│── notebooks/          # Jupyter notebooks (.ipynb)
│── sql/                # SQL queries (.sql)
│── images/             # Screenshots / charts
│── README.md
│── requirements.txt
```

---

## 🔍 Key Steps Performed

1. Data Cleaning and Preprocessing using Python
2. Exploratory Data Analysis (EDA)
3. SQL Queries to extract insights
4. Business Insights generation

---

## 📊 Key Insights

* Identified top-performing products and categories
* Analyzed regional sales performance
* Found trends in profit and discount impact
* Discovered high-revenue customer segments
